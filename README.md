# PokeDesk

# Uso

```
composer intall
npm install
php artisan migrate
php artisan app:poke 100
npm run dev
```

## Command artisan

Como parametro se puede entregar la cantidad de pokemons a buscar.
por defecto si no se entrega parametro se buscaran 100 pokemones.

```
php artisan app:poke 10
```

## Frontend React

```
npm install
npm run dev
```
